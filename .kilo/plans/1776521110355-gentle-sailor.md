# Plan: Merge branch antigravity to main

## Objetivo
Crear un PR desde la rama `antigravity` hacia `main` y hacer merge automáticamente.

## Pasos

1. **Crear PR**: Crear un pull request de `antigravity` a `main`
   ```bash
   gh pr create --title "Merge antigravity to main" --body "Merge branch antigravity with all new features: admin dashboard, kitchen orders, waitress, canchas, etc."
   ```

2. **Hacer merge del PR**: Mergear el PR creado a main
   ```bash
   gh pr merge --admin --merge
   ```

3. **Verificar**: Confirmar que el merge fue exitoso y que estamos en main
   ```bash
   git checkout main && git pull origin main
   ```

## Nota
- Hay archivos eliminados que estaban sin commitear (GEMINI.md, agentrules)
- El merge automático requiere permisos de escritura en la rama main

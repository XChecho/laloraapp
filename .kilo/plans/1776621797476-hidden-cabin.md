# Plan: Modal de Alerta (AlertModal)

## Objetivo
Crear un modal de alerta reutilizable que pueda ser llamado desde cualquier pantalla mediante Zustand, con soporte para tipos success y error.

---

## 1. Crear store `useAlertStore`

**Archivo:** `src/store/useAlertStore.ts`

```typescript
type AlertType = 'success' | 'error';

interface AlertData {
  type: AlertType;
  title: string;
  description: string;
}

interface AlertStore {
  alert: AlertData | null;
  showAlert: (data: AlertData) => void;
  hideAlert: () => void;
}
```

---

## 2. Crear componente `AlertModal`

**Archivo:** `src/components/modals/AlertModal.tsx`

### Características:
- **Tipos:** success (verde) y error (rojo)
- **Props del diseño:**
  - Borde verde claro / rojo claro
  - Background verde oscuro / rojo oscuro
  - Icono checkmark para success, X para error
  - Título (string, máx caracteres)
  - Descripción
- **Cierre:** 
  - Autocierre después de 5 segundos
  - Presionar fuera del modal
  - Presionar botón X

### Estilos:
- **Success:** border-green-200, bg-green-800
- **Error:** border-red-200, bg-red-800

---

## 3. Agregar a GeneralModals

**Archivo:** `src/components/modals/generalModals.tsx`

```tsx
import { AlertModal } from './AlertModal';

// Agregar en el return:
<AlertModal />
```

---

## 4. Uso

Desde cualquier pantalla:

```typescript
import { useAlertStore } from '@store/useAlertStore';

const { showAlert } = useAlertStore();

// Success:
showAlert({ type: 'success', title: 'Éxito', description: 'Acción completada' });

// Error:
showAlert({ type: 'error', title: 'Error', description: 'Algo salió mal' });
```

---

## 5. Notas
- El modal se cierra automáticamente después de 5 segundos usando setTimeout
- El cierre por fuera usa Modal con `onRequestClose`
- El cierre por X es un botón en la esquina superior derecha

---

# Plan: CRUD de Menú (Categorías y Productos)

## Objetivo
Crear un módulo completo en la sección de Admin para gestionar categorías y productos del menú, conectando con el backend NestJS.

---

## 1. Crear Actions del Menú

**Archivos:** `core/actions/menu.actions.ts`

### Funciones necesarias:
- `getCategoriesAction()` - Obtener todas las categorías
- `getCategoryByIdAction(id)` - Obtener categoría específica
- `createCategoryAction(data)` - Crear categoría
- `updateCategoryAction(id, data)` - Actualizar categoría
- `deleteCategoryAction(id)` - Eliminar categoría
- `getProductsByCategoryAction(categoryId)` - Obtener productos de una categoría
- `createProductAction(data)` - Crear producto
- `updateProductAction(id, data)` - Actualizar producto
- `deleteProductAction(id)` - Eliminar producto

### Tipos:
```typescript
interface Category {
  id: string;
  name: string;
  description?: string;
  icon: string;
  active: boolean;
  itemsCount: number;
  displayOrder: number;
}

interface Product {
  id: string;
  name: string;
  description?: string;
  price: number;
  categoryId: string;
  image?: string;
  available: boolean;
  modifiers?: Modifier[];
}

interface Modifier {
  name: string;
  options: string[];
  required: boolean;
  multiple: boolean;
}
```
